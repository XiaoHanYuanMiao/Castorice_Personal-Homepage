# Castorice 个人主页项目
一个基于HTML5构建的个性化个人主页模板，包含动态视频背景、主题下载功能和社交链接。

## 🚀 功能特性

- 🎥 **自适应视频背景**  
  支持不同设备尺寸自动切换视频源（M3U8 MP4等格式）
  
- 🎨 **主题下载模块**  
  提供主题包快速下载入口

- 📱 **响应式布局**  
  适配手机/平板/桌面端

- 🌈 **动态交互效果**  
  包含渐入动画和按钮悬浮特效

- 🔗 **社交整合**  
  预置B站/QQ群/酷安等平台链接
